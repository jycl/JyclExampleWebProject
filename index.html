This website is made with documentation that describes the basics to HTML, CSS and JavaScript.

While the source code is meant to be informative, the website is quite simple.

Here we have an unordered list created with:
  • Open with <ul>
  • Each list item is surrounded by <li> tags

An ordered list is created different instead with:
  1. Open with <ol>
  2. Each list item is still surrounded by <li> tags

Learn more about this project by clicking here. We create a link using the <a> tag and href="" attribute.
Note: Make sure you add the http:// or https:// (the protocol) at the beginning if it is a web address.

Note that a stands for anchor and href stands for hypertext reference.
